Learn Process of Website Development

Saat ini, saya ingin membuat website sejenis e-Commerce yang khusus menjual berbagai jenis kebaya, songket, dll. Harapannya website ini dapat menangani jahit online dimana user dapat membeli bahan kain sekaligus menjahit bahan kain yang sudah dibeli dengan mengisi form ukuran tubuh sebagai data valid saat bahan kain akan dijahit. Website ini akan menyediakan informasi tentang jenis-jenis kebaya, songket, dll.
Proses pembuatan website yang akan saya lakukan ialah pertama, saya akan melakukan requirement gathering. Selama requirement gathering, saya akan melakukan observasi terhadap owner yang saya tuju dan mengumpulkan data-data apa saja yang diperlukan dalam membangun website. Setelah semua data terkumpul, saya akan melakukan design prototype website yang akan saya bangun. Kemudian saya masuk ke tahap implementasi membuat kode program. Setelah implementasi selesai, website akan di-testing. Setelah itu akan dilakukan demo produk kepada owner untuk mendapatkan feedback terhadap website.
